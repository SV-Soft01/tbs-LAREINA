# Baloncesto

Pequeña aplicación web para mostrar estadísticas y manejo básico de datos relacionados con baloncesto.

## Estructura del proyecto

- `public/` - Archivos estáticos del sitio.
- `css/styles.css` - Estilos de la interfaz.
- `js/` - Lógica de la aplicación.
  - `admin.js` - Scripts para la vista de administrador.
  - `firebase-config.js` - Configuración de Firebase.
  - `public.js` - Funciones para la interfaz pública.
  - `stats-engine.js` - Motor de cálculo de estadísticas.

## Uso

1. Abrir `public/index.html` para la vista pública.
2. Abrir `public/admin.html` para la gestión de datos.

## Notas

- Se asume que el proyecto utiliza Firebase para almacenamiento o autenticación.
- Puedes adaptar los scripts y estilos según las necesidades de tu aplicación.
